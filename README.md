# Bahre Hassab Kemer (ባሕረ ሐሳብ)

A premium, interactive Ethiopian Orthodox Tewahedo Church Calendar and Date Converter.

## 🚀 How to Sync to Your GitHub

If you are using the **Gebeya Dala** platform and want to see this project in your own GitHub account:

1. **Link your GitHub**: Ensure your GitHub account is connected to the platform.
2. **Click the Sync Button**: Look for the **"Sync to GitHub"** button on the top right header of the editor interface (outside of this app).
3. **Select Repository**: You will be prompted to create a new repository or select an existing one.
4. **Push Changes**: Once selected, the platform will automatically push all the source code to your GitHub.

## ✨ Features

- **Accurate Bahre Hassab Logic**: Calculates all movable fasts and feasts (ጾመ ነነዌ, ዐቢይ ጾም, ትንሣኤ, etc.) for any given year (ዓመተ ምሕረት).
- **Interactive Calendar**: Monthly view with Amharic and English month/weekday names.
- **Yearly Holiday List**: Comprehensive list of religious holidays and fasts with filtering and search capabilities.
- **Date Converter**: Seamlessly convert between Gregorian (GC) and Ethiopian (EC) dates.
- **Premium Traditional UI**: A "Luxury Traditional" aesthetic with Glassmorphism and parchment-style elements.
- **Mobile Responsive**: Optimized for all devices from smartphones to desktops.

## 🛠 Tech Stack

- **React 19**
- **Vite**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Sonner** (Toasts)

## 📖 Usage

1. **Calendar**: View the current month and its specific holidays. Use the sidebar to change the year.
2. **Holidays & Fasts**: See the full year's schedule. Filter by type (Fast vs Feast) or search by name.
3. **Converter**: Easily switch between calendars for planning or reference.

## 🇪🇹 About the Project

This project aims to preserve and digitize the ancient wisdom of the Ethiopian calendar system (Bahre Hassab), making it accessible to the modern generation through a clean and intuitive interface.
